Illinois Web Components Toolkit Colors

Intro

This page showcases the Illinois Web Components Toolkit's semantic color variables and the Illinois color palette. The semantic colors are designed to be used across components, while the Illinois palette provides a set of standardized colors for branding and design consistency.

Naming and grouping

Here is the naming convention for semantic colors.

--ilw-color / --group-- / semantic-name

Example: --ilw-color / --blue-- / background → --ilw-color--blue--background

Default group: --ilw-color / background → --ilw-color--background
